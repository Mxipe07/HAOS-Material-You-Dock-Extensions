HAOS Material You Dock

Modernes Home Assistant UI im Material You Stil (Android inspiriert).

---

✨ Features

- Bottom Dock Navigation
- Swipe Tabs mit dynamischen Pills
- Flüssige Animationen
- Adaptive Breiten & Layout
- Cleanes Material Design

---

⚙️ Voraussetzungen

- Home Assistant
- button-card
- card-mod
- swipe-card

👉 Empfohlen:
Material You Theme oder eigenes Theme

---

📦 Installation

1. YAML Dateien in dein Dashboard einfügen
2. Entitäten anpassen
3. Fertig

---

📁 Dateien

- "01-bottom-dock.yaml" → Dock Navigation
- "02-swipe-tabs.yaml" → Tabs / Filterleiste
- "03-feature-cards.yaml" → UI Karten

---

🎨 Design

Dieses Projekt basiert auf einem Material You inspirierten Design
und ist für mobile Nutzung optimiert (Android & iOS).
